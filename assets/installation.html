Installer calculate Linux

Commencez par démarer sur votre média à l'aide du boot menu.

Quand le grub apparait, configurez votre disposition de clavier.

Après quelques secondes, Calculate est lancée. Si vous souhaitez lancer ssh voici la procédure:

  • Tapez /etc/init.d/sshd start
  • Obtenez votre IP avec ip a
  • Le mot de passe par défaut est root
Toutes les commandes de ce tutoriel sont à titre indicatif, il est INDISPENSABLE d'adapter ces commandes à vos envies / votre machine.
